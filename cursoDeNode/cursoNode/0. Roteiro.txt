5 - Engenharia de software
	- coesão e acoplamento
	- manutenibilidade do código 
	- camadas da aplicação (routes, controller, services, repositories, BD server)
	- routes - contém as rotas
	- controller - contém o controle (lógica) do que ocorrerá dentro do back para responder a requisição
	- services - lógica de negócio, onde se colocam as regras do tipo, não cadastrar venda sem cliente e sem produto válido.
	- repositories - contém o código que se conecta ao BD e executa as operações no BD.
	- BD server é o próprio servidor do BD que é um software preparado para interagir com outros softwares.
	- Login com middleware e CORS
	- Raw SQL, Query Builder, ORM (https://levelup.gitconnected.com/raw-sql-vs-query-builder-vs-orm-eee72dbdd275)
